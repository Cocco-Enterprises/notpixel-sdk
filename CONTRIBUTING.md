
# Contributing to NotPixel SDK

Thank you for your interest in contributing to NotPixel!

## How to Contribute

Since the source code is private, contributions are limited to:

### üêõ Bug Reports

1. Search [existing issues](https://github.com/notpixel-ai/sdk/issues) first
2. Use the [bug report template](https://github.com/notpixel-ai/sdk/issues/new?template=bug_report.md)
3. Include:
   - SDK version (`npm list notpixel`)
   - Node.js version
   - Minimal reproduction code
   - Expected vs actual behavior

### üí° Feature Requests

1. Check if the feature already exists in [docs](https://docs.notpixel.ai)
2. Search [existing requests](https://github.com/notpixel-ai/sdk/issues?q=is%3Aissue+label%3Aenhancement)
3. Use the [feature request template](https://github.com/notpixel-ai/sdk/issues/new?template=feature_request.md)

### üìñ Documentation

Found a typo or unclear documentation? Open an issue with the `documentation` label.

## Response Time

- **Critical bugs**: 24-48 hours
- **Regular bugs**: 3-5 business days
- **Feature requests**: Reviewed monthly
