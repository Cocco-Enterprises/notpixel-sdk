## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please email security@notpixel.ai with:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Any suggested fixes

We will respond within 48 hours and aim to release patches within 7 days for critical issues.

## Bug Bounty

We currently do not have a formal bug bounty program, but we recognize and thank security researchers who responsibly disclose vulnerabilities.
